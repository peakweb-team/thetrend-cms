# Media Component

Single image or video block for content.

## Schema Location
`src/components/shared/media.json`

## Fields

| Field | Type | Required | Localized | Description |
|-------|------|----------|-----------|-------------|
| `file` | media | No | No | Image or video file |

## Usage

Used in DynamicZone blocks to embed media within content.

## Supported Types
- Images (JPEG, PNG, GIF, WebP, SVG)
- Videos (MP4, WebM)

## Best Practices
- Use descriptive filenames for SEO
- Optimize images before upload
- Provide alt text via the media library
