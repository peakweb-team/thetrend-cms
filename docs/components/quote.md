# Quote Component

Quotation blocks with attribution.

## Schema Location

`src/components/shared/quote.json`

## Fields

| Field   | Type   | Required | Localized | Description              |
| ------- | ------ | -------- | --------- | ------------------------ |
| `title` | string | No       | Yes       | Quote author/attribution |
| `body`  | text   | No       | Yes       | Quote text               |

## Usage

Used in DynamicZone blocks for:

-   Testimonials
-   Pull quotes
-   Expert citations
-   Customer reviews

## Display

Frontend typically renders as a styled blockquote with author attribution below.
