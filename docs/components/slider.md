# Slider Component

Image carousel/gallery block.

## Schema Location
`src/components/shared/slider.json`

## Fields

| Field | Type | Required | Localized | Description |
|-------|------|----------|-----------|-------------|
| `files` | media | No | No | Multiple images |

## Usage

Used in DynamicZone blocks for:
- Image galleries
- Product showcases
- Before/after comparisons
- Portfolio displays

## Best Practices
- Use consistent image dimensions
- Optimize images for web
- Provide alt text for accessibility
